# KBD_Task-1

Project Summary: Spam Email Classifier
For this project, I developed a machine learning model to classify SMS messages as either 'spam' or 'ham' (legitimate).
I utilized the scikit-learn library to build a classification pipeline, employing TF-IDF Vectorizer for text feature extraction and a Multinomial Naive Bayes algorithm for prediction. 
After training on the SMS Spam Collection dataset, the model achieved an overall accuracy of 97%. 
This project provided me with foundational experience in natural language processing (NLP), data preprocessing, and model evaluation metrics like precision and recall.
